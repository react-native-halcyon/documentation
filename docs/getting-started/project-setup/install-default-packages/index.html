Install default packages

The necessary packages we need for a smooth transition of projects.

@react-native-community/async-storage

  • AsyncStorage is a simple, unencrypted, asynchronous, persistent, key-value storage system that is global to the app. It should be used instead of LocalStorage.

  • Used to store data persistently by using keys.

    e.g auth sessions

    • Install :

      yarn add @react-native-community/async-storage

    • Link :

      react-native link @react-native-community/async-storage

@react-native-community/netinfo

  • Used to check network status.

    • Install : 
      yarn add @react-native-community/netinfo@2.0.10

      Do not use version 3.1.0 yet! Cause it is broken.

frisbee

  • Used for API interactions

    • Install : 
      npm install --save frisbee

native-base

  • UI component package

    • Install :

      npm install native-base --save

    • Link :

      react-native link

prop-types

  • Used for prop validations

    • Install : 
      npm install --save prop-types

react-native-config

  • Used to read .env variables

    • Install : 
      yarn add react-native-config

      Keep in mind that this module doesn't obfuscate or encrypt secrets for packaging, so do not store sensitive keys in .env. It's basically impossible to prevent users from reverse engineering mobile app secrets, so design your app (and APIs) with that in mind.

react-native-fast-image

  • Used as an Image engine for our app. includes (image caching).

    • Install :

      yarn add react-native-fast-image

    • Link :

      react-native link react-native-fast-image

react-native-navigation

  • Used for navigation / router

    • Install :

      npm install --save react-navigation

      dependency

      react-navigation requires you to install react-native-gesture-handler for it to work.

      react-native-gesture-handler

      • Install :

        npm install --save react-native-gesture-handler

      • Link :

        react-native link react-native-gesture-handler

        To finalise installation of react-native-gesture-handler for Android, be sure to make the necessary modifications to android/app/src/main/java/com/app/MainActivity.java :

        package com.reactnavigation.example;

import com.facebook.react.ReactActivity;
+ import com.facebook.react.ReactActivityDelegate;
+ import com.facebook.react.ReactRootView;
+ import com.swmansion.gesturehandler.react.RNGestureHandlerEnabledRootView;

public class MainActivity extends ReactActivity {

  @Override
  protected String getMainComponentName() {
    return "Example";
  }

+  @Override
+  protected ReactActivityDelegate createReactActivityDelegate() {
+    return new ReactActivityDelegate(this, getMainComponentName()) {
+      @Override
+      protected ReactRootView createRootView() {
+       return new RNGestureHandlerEnabledRootView(MainActivity.this);
+      }
+    };
+  }
}

        See react-navigation installation for more info.

        Visit react-navigation docs to learn more about react-navigation.

react-native-keyboard-aware-scrollview

  • Used to handle keyboard appearance and automatically scroll to focused TextInput.

    • Install :

      yarn add react-native-keyboard-aware-scroll-view

    • Additional config for android : On Android android/app/src/main/Androidmanifest.xml change adjustResize to adjustPan

      <application
  android:name=".MainApplication"
  android:label="@string/app_name"
  android:icon="@mipmap/ic_launcher"
  android:roundIcon="@mipmap/ic_launcher_round"
  android:allowBackup="false"
  android:theme="@style/AppTheme">
  <activity
    android:name=".MainActivity"
    android:label="@string/app_name"
    android:configChanges="keyboard|keyboardHidden|orientation|screenSize"
+    android:windowSoftInputMode="adjustPan"
    android:launchMode="singleTop">
    <intent-filter>
        <action android:name="android.intent.action.MAIN" />
        <category android:name="android.intent.category.LAUNCHER" />
    </intent-filter>
  </activity>
  <activity android:name="com.facebook.react.devsupport.DevSettingsActivity" />
</application>

react-native-modal

  • Used to replace native Modal to a more customizable one.

    • Install : 
      yarn add react-native-modal

react-native-modal-dropdown

    • Install : 
      npm install react-native-modal-dropdown --save

react-native-vector-icons

  • Used as an Icon component.

    • Install :

      npm install react-native-vector-icons

    • Link :

      react-native link react-native-vector-icons

rn-content-loader

  • Used for our SkeletonLoader component.

      react-native link react-native-svg
